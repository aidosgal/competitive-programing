# Competitive Programming Solutions in C++

This repository contains my C++ solutions for various competitive programming problems from different platforms, including CodeForces and AtCoder. I use this repository to store and track my progress in problem-solving.

## Folder Structure

- `CodeForces/`
  - `RoundXXX/`
    - `ProblemA.cpp`
    - `ProblemB.cpp`
    - ...
  - `RoundYYY/`
    - `ProblemC.cpp`
    - `ProblemD.cpp`
    - ...

- `AtCoder/`
  - `ContestXXX/`
    - `problem_a.cpp`
    - `problem_b.cpp`
    - ...
  - `ContestYYY/`
    - `problem_c.cpp`
    - `problem_d.cpp`
    - ...

## How to Use

You can navigate through the folders based on the specific platform and contest to find the solutions for the respective problems. Each solution file includes the problem statement, my approach, and the corresponding C++ code. These solutions can be used for reference and learning purposes.

## Compiler and Environment

All solutions are implemented in C++ and compiled using g++ (GNU C++ Compiler) on a Unix-based system. Make sure you have a compatible compiler set up to execute the code.
